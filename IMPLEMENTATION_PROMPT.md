# 可复用的建站提示词

下面这段提示词已经按照当前网站目标整理。以后需要让 AI 修改或重建网站时，可以直接使用：

> 创建一个可直接部署到 GitHub Pages 的纯静态个人网站，只使用 HTML、CSS 和原生 JavaScript，不使用框架、后端、数据库和构建工具。网站视觉参考 daiqing-qi.github.io，但不得复制对方姓名、文章、照片、字体文件或品牌素材。首页采用左上角小图和少量文字入口；内页采用固定左侧导航、大面积留白、约 660px 内容列、黑白配色和等宽字体。网站包含 Writing 与 Photos 两个栏目。Writing 从独立 JavaScript 数据文件读取文章，并支持文章列表和单篇阅读。Photos 从独立 JavaScript 数据文件读取摄影系列，支持系列封面、相册详情、全屏灯箱、上一张/下一张、Escape 关闭、键盘方向键和移动端适配。所有路径必须兼容 GitHub 用户站点与项目站点。提供 GitHub Pages Actions 工作流、清晰的 README，以及无需编程即可新增照片和文章的操作说明。页面应可访问、响应式、加载快速，并尊重 prefers-reduced-motion。

推荐分类：`coding`、`web-development`、`portfolio`、`github-pages`、`photography`。
